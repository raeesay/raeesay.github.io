# raeesay.github.io

Personal website, served via GitHub Pages at https://raeesay.github.io

## Structure

- `index.html` — page content
- `style.css` — styling (light/dark mode via `prefers-color-scheme`)

## Editing

Just edit `index.html` directly — look for `<!-- TODO -->` comments marking
placeholder content (bio, research interests, publications, LinkedIn/Scholar
links) to fill in with your own details. No build step required.
